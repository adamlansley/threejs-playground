# three.js playground project
This project is intended to explore three.js, and see what can be achieved.

## Starting the project
- Clone the repository, making sure you're on latest
- Install all the relevant packages
- Run `pnpm run dev`
- Your terminal will describe a local url to visit to access the project

## Packages and tooling
- I've used WebStorm as my IDE
- ESLint and Prettier are configured to code quality
- Vite is the build tool used for bundling and building the application
- TypeScript is configured, inconjunction with relevant types for other packages
- three.js is the rendering engine of choice
